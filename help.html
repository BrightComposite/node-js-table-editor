Кнопка в левом верхнем углу - для открытия меню импорта-экспорта

Контекстное меню для изменения строк таблицы открывается при нажатии на кнопку в первой ячейки нужной строки

A - добавить строку сверху

B - добавить строку снизу

D - дублировать текущую строку

X - удалить строку

Строки можно перемещать при помощи мыши

Использованы следующие компоненты:

jQuery (v1.11.3), лицензия MIT

jQuery UI (v1.12.0), лицензия MIT

Репозиторий с исходным кодом: GitHub
